CSS Transitions

Denne blir kul når du fører musa over

#demoTransition {
font-size: 12px;
color: #000;
-o-transform: rotate(0deg);
-o-transition-property: font-size, color;
-o-transition-duration: 0.5s;
line-height: 100px;
}

#demoTransition:hover {
font-size: 30px;
color: #ff0000;
-o-transform: rotate(-5deg);
}
